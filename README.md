# sandbox

This repo is a skills test using React, TypeScript, and Jest.

## Prerequisites

- Node v10.16.0 or higher

## Running the software

```bash
git clone https://github.com/bshawn/sandbox.git
cd ./sandbox/web-client
npm install
npm start
```

## Running the tests

```bash
npm test
```
